## Success Criteria - 90-Day Checkpoint

By the end of 90 days, the new team member should be able to:

- [ ] Independently conduct security risk assessments
- [ ] Understand system architecture and security controls
- [ ] Contribute to RMF activities and authorization process
- [ ] Develop and track remediation plans
- [ ] Participate in design reviews and provide security recommendations
- [ ] Monitor compliance with security requirements
- [ ] Communicate security findings to technical and executive audiences
- [ ] Effectively collaborate across organizational functions
